Sou a Marta, tenho 25 anos de idade, sou estudante de Análise e Desenvolvimento de sistemas e gosto muito de aprender coisas novas para desenvolver as minhas habilidades. Meu interesse por programação começou em 2019, quando comecei a assistir alguns vídeos no YouTube sobre o assunto e fiquei apaixonada então fui atrás de cursos gratuitos. Atualmente estou focada no aprendizado de HTML, CSS, JAVASCRIPT, Mongo, SQLITE, API REST, NODE JS

Em 2020 fiz o meu primeiro curso ao participar do projeto ProgaMaria onde construi o meu primeiro projeto, que foi um site para suprender o meu esposo falando da vinda da nossa filha ao mundo, usando HTML e CSS. Estou sempre procurando aprender coisas novas. Atualmente estou a fazer um curso intesivo na instituição de ensino Toti onde estou vendo HTML, CSS3, JAVASCRIPT, Mongo, SQLITE, API REST e NODE JS. Tenho como lema da minha vida "lutar sempre, vencer se for possível mas desistir nunca", nunca desisto dos meus obtivos e ser programadora é um dos princípais.
